## 1. 扩展程序用途

Color Analyzer for YouTube 是一款 Chrome 扩展程序，可根据 YouTube 视频的可见画面在本地生成分量图、波形图和矢量示波器。其唯一用途是观察和分析色彩；它不会修改视频、视频源文件或视频的呈现效果。

## 2. 摘要

- 只有在用户明确点击扩展程序图标，并在首次使用或数据披露更新后接受扩展程序内显示的数据披露时，分析才会开始；
- 页面上下文和播放器状态仅在分析会话处于活动状态时被观察，并会随该会话结束而停止；
- 可见的视频像素在设备本地处理；
- 不捕获音频；
- 不会将任何视频图像保存到磁盘或发送给发布者；
- 扩展程序不设用户账户，不含广告、受众分析工具或应用服务器；
- 发布者不会出售、共享或接收分析产生的任何数据。

## 3. 处理的数据

### 3.1 可见的视频像素

在分析会话处于活动状态期间，扩展程序会临时捕获所选 YouTube 标签页的完整视觉输出。随后，它会裁剪出可见的视频区域，并且只读取该区域内的像素值，以计算三种色彩分析示波器。

捕获的画面可能包含叠加在视频上的可见元素，例如字幕或播放器控件。扩展程序会对其中某些情况发出提示，因为这些元素可能影响测量结果。

原始像素数组只会在计算所需期间保留在工作内存中，随后其引用会被释放。在活动会话期间，本地画布可能会在内存中保留最后一幅裁剪图像，直到它被另一幅图像替换。捕获停止时，画布会重置为 1 × 1 像素并释放视频源。任何图像都不会写入持久存储、添加到历史记录或通过互联网传输。

### 3.2 页面上下文和播放器状态

扩展程序的本地脚本会在 `youtube.com` 页面上运行，但在用户接受当前数据披露并明确启动分析之前，它会保持不活动。仅在分析会话处于活动状态时，它才会定期观察页面上下文和播放器状态。当用户点击“停止”、导航到不受支持的页面、关闭标签页或捕获结束时，此观察会立即停止。关闭侧边栏后，它会在一段用于容许侧边栏重新加载的短暂技术宽限期之后停止。两次分析会话之间不会观察播放器上下文。为了正确定位视频、检测 YouTube 的站内导航并在活动会话期间同步测量，扩展程序会临时处理：

- 当前 YouTube 页面的地址和视频标识符；
- 播放时间以及正在播放、已暂停或正在跳转的状态；
- 播放器模式、标签页可见性，以及控件或字幕是否可见；
- 窗口、播放器和视频自身的尺寸。

这些信息仅用于提供用户请求的分析、在无法可靠测量源画面时暂停测量计算，以及避免分析错误的区域。页面地址、视频标识符和播放时间不会被持久保存，也不会发送给发布者。

### 3.3 本地偏好设置

扩展程序会在 Chrome 本地存储中保存以下显示偏好设置：选中的示波器、分量图模式、波形图通道、彩色显示设置以及肤色线是否显示。它还会保存用户已接受的数据披露版本。该技术值不包含身份信息、页面地址或视频图像。

这些偏好设置会保留在设备上，直至被替换、扩展程序数据被清除或扩展程序被卸载。

### 3.4 技术会话数据

在浏览器会话期间，扩展程序可能会保留一个随机会话标识符、被捕获标签页的内部标识符以及最新的分析状态。这些信息仅用于将测量结果与正确的捕获关联起来，并正常停止捕获。它们保留在 Chrome 会话存储中，不会发送给发布者。

## 4. 传输、共享和出售

Color Analyzer for YouTube 不会向发布者或任何第三方传输用户数据。页面脚本、屏幕外文档、Web Worker、Service Worker 与侧边栏之间的消息传递始终在设备上的扩展程序内部进行。

本扩展程序：

- 不出售任何数据；
- 不为广告、用户画像或信用评估目的共享数据；
- 不会将数据用于与色彩分析无关的任何用途；
- 不执行远程托管的代码；
- 不包含任何遥测或受众分析系统。

用户使用 YouTube 和 Google 的服务时，这些服务可能会独立处理数据。此类处理受它们各自的政策约束，不由本扩展程序控制。

## 5. 保留和删除

- **视频像素**：本地工作内存；原始数组会在计算后释放。最后一幅裁剪图像只可能在活动会话期间保留在画布中；停止时，画布会重置为 1 × 1 像素并释放视频源。
- **播放器上下文**：临时内存，仅在分析会话处于活动状态时持续被替换。观察不会在同意前开始，并会在会话结束时立即停止。
- **会话状态**：捕获停止时会移除活动捕获标识符；最新状态可能保留在 Chrome 会话存储中，直至浏览器会话结束。
- **显示偏好设置和同意版本**：保存在 Chrome 本地存储中，直至被更改、清除或扩展程序被卸载。

点击“停止”、导航到不受支持的页面、关闭标签页或捕获结束时，捕获、像素分析和播放器上下文观察会立即停止。关闭侧边栏会在一段用于容许侧边栏重新加载的短暂技术宽限期之后触发相同的清理。此清理会将分析画布重置为 1 × 1 像素并释放视频源。同意前以及活动分析会话结束后都不会观察播放器上下文。用户可以通过清除 Chrome 中的扩展程序数据或卸载扩展程序来删除已保存的偏好设置。

发布者没有这些信息的远程副本，因此无法远程访问或删除这些信息。

## 6. Chrome 权限

扩展程序仅使用实现其用途所需的权限：

- **tabCapture**：临时捕获所选标签页的可见输出，不包含音频；
- **offscreen**：在 Chrome 屏幕外文档中接收并在本地分析捕获的媒体流；
- **sidePanel**：在 Chrome 侧边栏中显示各示波器及其控件；
- **storage**：保留本地偏好设置、同意版本和技术会话状态；
- **访问 `https://www.youtube.com/*`**：仅在分析会话处于活动状态时检测 YouTube 播放器、其几何位置、状态以及是否已导航离开所选视频。只有在用户同意并执行操作后，捕获才会在兼容的 `/watch` 页面上开始。

## 7. 安全

数据处理隔离在扩展程序的本地组件中。其内容安全政策仅允许运行扩展程序包中内置的脚本。捕获的数据不会通过网络传输。

## 8. 有限使用合规性

对从 Google API 获取的信息的使用将遵守 Chrome 应用商店用户数据政策，包括有限使用要求。
