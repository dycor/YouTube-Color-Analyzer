## 详细说明

YouTube 色彩分析器使用当前播放视频的可见像素，在 Chrome 侧边栏中显示色彩分析示波器。

它专为希望在自己的项目中重现相似视觉风格之前研究参考视频的剪辑师、调色师和创作者而设计。

数据处理

在 YouTube 页面上，扩展程序会在本地读取页面地址、视频标识符和播放器状态，以检测并框定兼容的视频。在您接受扩展程序内的数据披露并开始分析后，它会临时捕获当前标签页的视觉输出（不含音频），然后裁剪视频区域以计算各示波器。

像素、页面地址和播放状态始终留在您的浏览器中。任何图像或播放数据都不会发送到服务器、发布者或任何第三方。显示偏好设置和同意版本会保存在 Chrome 本地存储中。捕获标识符和最新技术状态会临时保存在会话存储中，并最迟在浏览器会话结束时消失。

包含的示波器

• YRGB 或 RGB 分量图，采用标准化的 0–100 刻度。  
• 波形图，可分别选择 Y、R、G 和 B 通道，并可采用彩色或单色显示。  
• 基于 Rec.709 的矢量示波器，带有色相目标和可选的肤色参考线。

使用方式

打开标准的 YouTube 视频页面，点击扩展程序图标，并接受首次使用时显示的数据披露。侧边栏随即打开，并开始分析所选标签页中可见的视频。

视频播放期间，各示波器会实时更新。视频暂停时，扩展程序会分析一帧更详细的画面。三个示波器均根据同一帧画面计算。侧边栏不会遮挡视频，所选设置会保存在本地。

隐私

所有计算都在设备上进行。视频像素只存在于本地工作内存中；最后一幅裁剪图像可能会保留到被替换或屏幕外文档被销毁。任何图像都不会被持久保存、导出或发送到服务器。扩展程序没有后端、广告或受众分析工具，也不会捕获音频。

兼容性

• Google Chrome 116 或更高版本。  
• 标准 `youtube.com/watch` 页面。  
• 普通模式和影院模式。  
• 界面支持英语、法语、中文、西班牙语和葡萄牙语。

版本 1 的限制

扩展程序分析的是 Chrome 呈现的 SDR 输出，而不是原始视频文件或源信号。因此，标准化的 0–100 数值并非经过校准的广播级测量值。

不支持经过校准的 HDR 分析、Shorts、YouTube Music、嵌入式播放器、全屏、YouTube 迷你播放器和画中画模式。

字幕和叠加在可见视频上的其他元素可能影响测量结果。当 YouTube 播放器控件可见时，分析会暂停。

扩展程序仅提供测量示波器。它不会修改视频、应用色彩校正或提供自动调色建议。

## 建议的屏幕截图说明文字

1. **直接在 Chrome 中使用三种色彩分析示波器**  
   在专用侧边栏中使用 YRGB/RGB 分量图、波形图和矢量示波器。
2. **播放时实时分析**  
   随着视频播放，观察亮度、RGB 通道和饱和度。
3. **暂停时获得更详细的画面**  
   定格参考画面，更精确地查看其色彩分布。
4. **调色师熟悉的控件**  
   YRGB 通道、彩色或单色显示，以及可选的肤色线。
5. **本地、非破坏性处理**  
   不上传任何图像，也不将图像保存到持久存储中；视频绝不会被修改。
