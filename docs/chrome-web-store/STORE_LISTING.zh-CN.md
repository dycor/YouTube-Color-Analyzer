## 详细说明

Color Analyzer for YouTube 使用当前播放视频的可见像素，在 Chrome 侧边栏中显示色彩分析示波器。

它专为希望在自己的项目中重现相似视觉风格之前研究参考视频的剪辑师、调色师和创作者而设计。

数据处理

只有在您接受扩展程序内当前的数据披露并明确启动分析之后，扩展程序才会在本地读取用于检测和框定兼容视频的 YouTube 页面地址、视频标识符和播放器状态。此上下文仅在分析会话处于活动状态时被观察。扩展程序会临时捕获当前标签页的视觉输出（不含音频），然后裁剪视频区域以计算各示波器。

点击“停止”、关闭侧边栏、导航到其他页面或捕获结束时，上下文观察和捕获都会立即停止。像素、页面地址和播放状态始终留在您的浏览器中。任何图像或播放数据都不会发送到服务器、发布者或任何第三方。显示偏好设置和同意版本会保存在 Chrome 本地存储中。捕获标识符和最新技术状态会临时保存在会话存储中，并最迟在浏览器会话结束时消失。

包含的示波器

• YRGB 或 RGB 分量图，采用标准化的 0–100 刻度。  
• 波形图，可分别选择 Y、R、G 和 B 通道，并可采用彩色或单色显示。  
• 基于 Rec.709 的矢量示波器，带有色相目标和可选的肤色参考线。

使用方式

打开标准的 YouTube 视频页面，点击扩展程序图标，并接受首次使用时显示的数据披露。侧边栏随即打开，并开始分析所选标签页中可见的视频。

视频播放期间，各示波器会实时更新。视频暂停时，扩展程序会分析一帧更详细的画面。三个示波器均根据同一帧画面计算。侧边栏不会遮挡视频，所选设置会保存在本地。

隐私

所有计算都在设备上进行。视频像素仅在分析会话处于活动状态时存在于本地工作内存中。分析停止时，视频源会被释放，画布会重置为 1 × 1 像素。任何图像都不会被持久保存、导出或发送到服务器。扩展程序没有后端、广告或受众分析工具，也不会捕获音频。

兼容性

• Google Chrome 116 或更高版本。  
• 标准 `youtube.com/watch` 页面。  
• 普通模式和影院模式。  
• 界面支持英语、法语、中文、西班牙语和葡萄牙语。

版本 1 的限制

扩展程序分析的是 Chrome 呈现的 SDR 输出，而不是原始视频文件或源信号。因此，标准化的 0–100 数值并非经过校准的广播级测量值。

不支持经过校准的 HDR 分析、Shorts、YouTube Music、嵌入式播放器、全屏、YouTube 迷你播放器和画中画模式。

字幕和叠加在可见视频上的其他元素可能影响测量结果。当 YouTube 播放器控件可见时，分析会暂停。

扩展程序仅提供测量示波器。它不会修改视频、应用色彩校正或提供自动调色建议。
